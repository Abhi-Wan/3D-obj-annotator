# 3D Model Viewer and Annotator

Upload a 3D model file (.glb, .gltf, .obj, or .usdz) to view and annotate with arrows (more options to come). Save screenshots of the model with the annotations at various resolutions, and view your session screenshots on the Gallery page.

Steps to run:
1. Clone repo
2. ```npm install```
3. ```npm run dev```
